# 🚀 MERN Foundation Exercises

Welcome to the **MERN Foundation Exercises** repository! This is a structured, 30-day intensive journey designed to take you from JavaScript fundamentals to building a full-stack Task Manager application.

## 📌 Project Overview

This repository tracks daily progress through the four pillars of the MERN stack:

1.  **M**ongoDB (Database)
2.  **E**xpress.js (Backend Framework)
3.  **R**eact (Frontend Library)
4.  **N**ode.js (Runtime Environment)

---

## 🗓 30-Day Roadmap & Workflow

### 🧠 WEEK 1: JavaScript + Backend Thinking

_Focus: Master the logic before the tools._

- [x] **Day 1:** JS Recap (Variables, Scope, Functions) | _Task: 5 Utility functions._
- [x] **Day 2:** Array Methods (Map, Filter, Reduce) | _Task: Student Score Analyzer._
- [ ] **Day 3:** Objects & Nesting | _Task: User Profile Manipulation._
- [ ] **Day 4:** Destructuring & Spread/Rest | _Task: Object Merge/Clone logic._
- [ ] **Day 5:** Async JS (Callbacks & Promises) | _Task: API Delay Simulator._
- [ ] **Day 6:** Async/Await & Try/Catch | _Task: Fake API Handler._
- [ ] **Day 7:** **Mini Review** | _Build: User Data Processor CLI._

### ⚙️ WEEK 2: Node.js + Express Basics

_Focus: Learning how the server speaks to the web._

- [ ] **Day 8:** Node.js Runtime & `http` module.
- [ ] **Day 9:** Express Setup & Basic Routing (GET/POST).
- [ ] **Day 10:** Middleware | _Task: Custom Request Logger._
- [ ] **Day 11:** REST API Structure (In-memory CRUD).
- [ ] **Day 12:** Postman Mastery (API Testing).
- [ ] **Day 13:** Centralized Error Handling.
- [ ] **Day 14:** **Mini Project** | _User Management API (In-memory)._

### 🗄️ WEEK 3: MongoDB + Mongoose

_Focus: Persistence and data modeling._

- [ ] **Day 15:** MongoDB Basics (Collections & Documents).
- [ ] **Day 16:** Mongoose Setup & Schema Design.
- [ ] **Day 17:** CRUD operations with MongoDB.
- [ ] **Day 18:** Schema Validation & Constraints.
- [ ] **Day 19:** Advanced Querying (Sort, Limit, Filter).
- [ ] **Day 20:** Integrating Express with MongoDB.
- [ ] **Day 21:** **Mini Project** | _User API with Database Persistence._

### ⚛️ WEEK 4: React + Full MERN Integration

_Focus: Bringing the UI to life and connecting the dots._

- [ ] **Day 22:** React Basics (JSX & Components).
- [ ] **Day 23:** State & Props Management.
- [ ] **Day 24:** `useEffect` & Fetching Data.
- [ ] **Day 25:** Form Handling & Controlled Components.
- [ ] **Day 26:** Connecting React to Backend API (CORS).
- [ ] **Day 27:** Rendering Dynamic Backend Data.
- [ ] **Day 28:** Full CRUD Frontend Integration.
- [ ] **Day 29:** **🚀 Capstone Project** | _MERN Task Manager (Full Features)._
- [ ] **Day 30:** Clean Code, Deployment Logic (Render/Vercel) & Review.
